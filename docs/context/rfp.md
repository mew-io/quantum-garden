# Qollab RFP - Quantum Computing Experiments

_This document contains information about the Qollab Request for Proposals that Quantum Garden was submitted to._

---

## About Qollab

Qollab supports experimental projects at the intersection of quantum computing and creative practice. Their quantum computing experiments program provides funding and IonQ compute credits to selected projects.

Website: https://qollab.xyz

---

## Program Overview

The quantum computing experiments program is designed to support:

- Creative applications of real quantum hardware
- Open source projects that benefit the community
- Explorations that make quantum computing more accessible
- Work that doesn't rely solely on simulation

---

## Requirements

Projects must:

1. **Use Qiskit framework** — The standard framework for quantum circuit design
2. **Be open source** — Code published publicly on GitHub
3. **Use IonQ hardware** — Compute credits provided for real quantum execution
4. **Deliver on timeline** — Kickoff to final delivery within program window

---

## Support Provided

- **Funding**: Up to $5,000 USD for project development
- **Compute**: Up to $15,000 in IonQ compute credits
- **Community**: Connection to Qollab's network of quantum practitioners

---

## Timeline

- **Kickoff**: January 15
- **Initial delivery**: February 23
- **Final milestone**: March 2

---

## Selection Criteria

Projects are evaluated on:

- Technical feasibility with current quantum hardware
- Creative and educational value
- Quality of open source contribution
- Team capability and experience
- Clarity of project scope and deliverables

---

## Quantum Garden's Fit

Quantum Garden aligns with the RFP by:

1. Using Qiskit for all quantum circuit generation
2. Publishing everything as open source
3. Running circuits on real IonQ hardware (not just simulation)
4. Delivering a polished, documented web experience
5. Making quantum concepts accessible through metaphor

The project demonstrates how quantum computing can inform creative work in a meaningful, non-gimmicky way.
